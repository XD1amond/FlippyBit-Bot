# FlippyBit-Bot
Automated Flippy Bit Script

## Instructions
1. Download Main.py
2. Run script
   
### Options
Press "Q/q" at any time during the script to end it. When game is complete, press Y/N to play again or not respectively.  

## Requirements
- Python 3.12
- Selenium library
- Firefox, Chrome, Edge, or Safari
- ChromeDriver (If using Chrome)
